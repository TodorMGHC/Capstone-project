# Capstone Project

A modular Vite single-page app scaffold with routed pages, reusable components, and Bootstrap styling.

## Scripts

- `npm run dev` - start the Vite dev server
- `npm run build` - build for production
- `npm run preview` - preview the production build

## Routes

- `/` - empty home page with "Hello, world!"
- `/login` - login page scaffold
- `/dashboard` - dashboard page scaffold
- `/app/:id` - dynamic app page
